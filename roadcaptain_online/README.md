# Road Captain by Rhegium - Google V3.1.2

Correzioni:
- colori nero/bianco/rosso
- font suggerimenti più piccolo e non bold
- calendario forzato tramite pulsante
- prezzo carburante funzionante senza chiave Google
- calcolo con fallback gratuito se GOOGLE_MAPS_API_KEY non è configurata
- supporta anche GOOGLE_MAPS_KEY o MAPS_API_KEY come nomi variabile

Render:
Root Directory: roadcaptain_online
Build Command: npm install
Start Command: node server.js

Environment consigliato:
GOOGLE_MAPS_API_KEY=<chiave>
