# Road Captain by Rhegium - Google Engine V3.1

Web app online-first per pianificazione viaggi moto/auto.

## Deploy Render
Root Directory: `roadcaptain_online`
Build Command: `npm install`
Start Command: `node server.js`
Environment: `GOOGLE_MAPS_API_KEY`

## V3.1
- Partenza/arrivo vuoti con placeholder.
- Data selezionabile con calendario HTML nativo.
- Analisi testo libera per compilare i campi principali.
- Prezzo carburante aggiornabile da MIMIT prezzo medio SELF alle 8, con fallback manuale dichiarato.
- Google Geocoding/Routes/Places per motore principale.
