# Road Captain by Rhegium - Online First V2

Fix motore soste:
- pre-carica i POI carburante lungo il corridoio della rotta;
- seleziona soste con logica 150 -> 175 km -> ritroso;
- non propone deviazioni come soste valide;
- corregge orario di arrivo/partenza senza slittamento fuso orario;
- espone conteggio POI carburante rilevati sulla rotta.

Render:
- Root Directory: roadcaptain_online
- Build Command: npm install
- Start Command: node server.js
